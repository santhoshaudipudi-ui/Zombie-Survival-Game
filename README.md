# Zombie Survival: The Last Survivor

## About the Project

Zombie Survival: The Last Survivor is a Python-based survival game developed as my final project for the Stanford Code in Place 2026 program.

Being selected for Code in Place 2026 was an exciting opportunity to learn programming from Stanford's curriculum and global community. As I progressed through the program and reached the final project phase, I wanted to apply the concepts I learned—graphics, user interaction, loops, conditionals, lists, and game logic—to create an engaging and interactive game.

This project represents my learning journey throughout Code in Place and demonstrates how programming concepts can be combined to build a complete game experience.

## Description

In Zombie Survival: The Last Survivor, the player controls a survivor trapped in a zombie-infested area. The objective is to collect all supply boxes while avoiding zombies that continuously chase the player.

The game combines survival mechanics, real-time movement, collision detection, scoring, and win/lose conditions to create an enjoyable gameplay experience.

## Features

* Arrow key player movement
* Zombie chasing AI
* Health system
* Score system
* Supply collection mechanics
* Win screen
* Game Over screen
* Interactive graphical interface

## How to Play

1. Use the Arrow Keys to move the player.
2. Collect all yellow supply boxes to increase your score.
3. Avoid red zombies that chase you.
4. Survive and collect all supplies to win.
5. If your health reaches zero, the game ends.
